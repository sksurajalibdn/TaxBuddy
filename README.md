# TaxBuddy
Fill your ITR with a Swag like a CA 

Installation (3 steps)
1. Make sure Python 3.9+ is installed

python3 --version
If not installed: download from python.org/downloads

2. Unzip and install

unzip TaxBuddy.zip
cd TaxBuddy
pip install .
3. Run

taxbuddy
Opens in browser at http://localhost:8501

If pip install . has issues, the fallback is:

pip install streamlit pdfplumber openpyxl
python -m streamlit run taxbuddy/app.py
The zip includes a detailed README.md with troubleshooting, feature overview, and document download instructions. Just share the zip file via DM, email, or Google Drive.
